# 💣 Bomb Party - Single Player

Uma versão minimalista do clássico jogo de palavras "Bomb Party", desenvolvida para um único jogador testar seu vocabulário e velocidade sob pressão.

---

## 🎮 Como Funciona

O objetivo é simples: o jogo te dá uma **sílaba** e você precisa digitar uma **palavra** que contenha essa sequência de letras antes que o tempo acabe.

* **Vidas:** Você começa com 3 corações. O tempo acabou? Perde uma vida.
* **Tempo:** O cronômetro é imprevisível, variando entre 4 a 8 segundos.
* **Dicionário:** O jogo valida se a palavra existe no arquivo `words.json`.
* **Recordes:** Suas 5 melhores pontuações ficam salvas no navegador (LocalStorage).

* **Validação em Tempo Real:** Verifica se a palavra já foi usada ou se contém a sílaba solicitada.

# 💣 Bomb Party - Single Player

A minimalist version of the classic word game "Bomb Party," developed for a single player to test their vocabulary and speed under pressure.

---

## 🎮 How It Works

The goal is simple: the game gives you a **syllable** and you need to type a **word** that contains that sequence of letters before time runs out.

* **Lives:** You start with 3 hearts. Made a mistake or ran out of time? You lose a life.
* **Time:** The timer is unpredictable, ranging between 4 and 8 seconds.
* **Dictionary:** The game validates if the word exists in the `words.json` file.
* **Records:** Your top 5 scores are saved in the browser (LocalStorage).

---

## ✨ Features

* **Dark Interface:** Modern design with a *glassmorphism* effect.
* **Real-time Validation:** Checks if the word has already been used or if it contains the requested syllable.
* **Ranking:** Automatic Highscore system.

---

## 🛠️ Tech Stack

* **HTML5** & **CSS3**
* **JavaScript (ES6+)**
* **Bootstrap Icons** (for the life hearts)
* **JSON** (word database)

---

## 📂 Folder Structure

```text
Bomb-Party_single-player/
├── public/
│   ├── index.html    # Page structure
│   ├── script.js     # Game logic (Syllables, Timer, Scores)
│   └── style.css     # Styling and layout
├── words.json        # Dictionary of allowed words
└── README.md         # Project documentation
```

## 🚀 How to Run the Project

1. Clone or download the files.
2. Ensure the `words.json` file is in the root directory according to the structure.
3. Make sure you have the **Live Server** extension.
4. Open the `index.html` file and click **Go Live**.
